Guia de uso do markdown

Ênfase

**negrito**
*itálico*

Títulos

# Título 1
## Título 2
### Título 3
#### Título 4

Listas

* Tópico
* Tópico
* Tópico

1. Item
2. Item
3. Item

Links

[Texto exibido](http://www.example.com)

Citações

> Esta é uma citação.
> Ela pode ocupar várias linhas!

Imagens

![](http://www.example.com/image.jpg)

Tabelas

| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| John     | Doe      | Male     |
| Mary     | Smith    | Female   |

Não é necessário alinhar as colunas...

| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| John | Doe | Male |
| Mary | Smith | Female |

Código

Uso inline...

`var example = "hello!";`

Um bloco de código...

```
var example = "hello!";
alert(example);
```
